# symptomchecker
<a href="http://priaid.com">priaid.com</a> (under the subbrand <a href="http://apimedic.com">ApiMedic.com</a>) offers a medical symptom checker primarily for patients. Based on the entered symptoms it tells you what possible diseases you have. It directs you to more medical information and shows you the right doctor for further clarifications. The symptom checker can be integrated via the flexible API (Application Programming Interface). Find here code examples (SDK).
